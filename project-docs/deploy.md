# Deploy to GitHub Pages

## Workflow

1. Edit service pages in **`pages/`** (source of truth)
2. Edit blog directly in **`blog/`**
3. Push to `main` — GitHub Action runs `publish.py` and deploys automatically

For local preview without publishing:

```bash
python server.py    # http://localhost:8080
```

After editing CSS in `assets/css/`, rebuild the bundle:

```bash
python scripts/bundle-css.py
```

Fonts are self-hosted in `assets/fonts/`. Re-download if needed: `python scripts/fetch-fonts.py`

Optimize images after adding new assets:

```bash
python scripts/optimize-images.py
```

To test the full build locally:

```bash
python scripts/publish.py
python -m http.server 8766
```

## GitHub Pages setup

1. Push repo to GitHub
2. **Settings → Pages → Build and deployment → Source:** GitHub Actions
3. Add custom domain: `www.givsharifi.com`
4. Create `CNAME` file in repo root: `www.givsharifi.com`
5. Configure Cloudflare (see `cloudflare-setup.md`)
6. Submit `sitemap.xml` in Google Search Console

## File structure

| Path | Role |
|------|------|
| `index.html` | Homepage |
| `pages/` | Service pages + done + 404 (source) |
| `blog/` | Blog (root, edit directly) |
| `brain-surgery/`, … | Generated by `publish.py` (gitignored) |
| `assets/`, `components/`, `posts/` | Shared static assets |
